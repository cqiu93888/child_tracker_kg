# 方法二：批次註冊教學（一次註冊多張照片）

適用情境：你已經把多張幼兒照片放在 `data/registered/`（例如 `iid_1.png`～`iid_16.png`），想一次幫每張照片指定名字，不用手動打 16 次指令。

---

## 步驟一：修改名字清單

有兩種方式可選，**擇一使用**即可。

### 方式 A：用 Python 腳本（建議，較不受系統限制）

1. 用記事本或 Cursor 開啟專案裡的 **`register_all.py`**。
2. 找到這一段：
   ```python
   names = [
       "小孩1", "小孩2", "小孩3", ...
   ]
   ```
3. 把 `"小孩1"`、`"小孩2"` … 改成每位幼兒的**真實名字**（順序要對應 `iid_1.png`、`iid_2.png`…）。
   - 例如：`iid_1.png` 是小明 → 第一個就填 `"小明"`。
4. 存檔。

### 方式 B：用 PowerShell 腳本

1. 用記事本或 Cursor 開啟專案裡的 **`register_all.ps1`**。
2. 找到這一段：
   ```powershell
   $names = @(
       "小孩1", "小孩2", "小孩3", ...
   )
   ```
3. 同樣把 `"小孩1"` 等改成真實名字，順序對應 `iid_1.png`、`iid_2.png`…。
4. 存檔。

---

## 步驟二：執行批次註冊

在專案資料夾**開啟終端機**（PowerShell 或命令提示字元），然後：

### 若你選「方式 A」（Python）

在專案目錄下執行：

```bash
cd "C:\Users\user\OneDrive - 淡江大學\Desktop\child_tracker_kg"
python register_all.py
```

看到「已註冊: 名字 <- iid_N.png」就表示該張照片已註冊成功。

### 若你選「方式 B」（PowerShell）

在專案目錄下執行：

```powershell
cd "C:\Users\user\OneDrive - 淡江大學\Desktop\child_tracker_kg"
.\register_all.ps1
```

若出現 **「無法載入檔案，因為這個系統上已停用指令碼執行」**：

1. 先執行（只需做一次）：
   ```powershell
   Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
   ```
2. 出現詢問時輸入 `Y` 並按 Enter。
3. 再執行一次：`.\register_all.ps1`。

## 步驟三：處理影片與建圖

註冊完成後，依序執行：

```bash
python main.py process --video data/input/test2.mp4 --output data/output/result.mp4
python main.py build-graph --output-dir data/graph
```

- 帶名字的影片會出現在 **`data/output/result.mp4`**。
- 關係圖請用瀏覽器開啟 **`data/graph/relationship_graph.html`**。

---

## 常見問題

- **照片不是 iid_1～iid_16 或數量不同？**  
  請打開 `register_all.py`，依你的檔名與數量修改 `names` 串列長度，以及裡面的 `photo = os.path.join(base, f"iid_{i}.png")`（改成你的檔名規則）。

- **只想註冊其中幾張？**  
  在腳本裡把不需要的註解掉，或只保留要註冊的編號與名字即可。
